# Python PyQt6 全屏图片播放器项目

## 课程介绍

PyQt 是 Python 和 Qt 的融合，是一个非常强大的 GUI 库。

本课程使用 Python PyQt6，带大家一步一步实现一个全屏图片播放器。

课程的主要内容有：

1. 项目演示；
2. 设计播放器主界面；
3. 实现选择文件夹；
4. 实现全屏自适应播放图片；
5. 实现暂停播放、继续播放；
6. 实现退出播放；
7. 程序发布。
